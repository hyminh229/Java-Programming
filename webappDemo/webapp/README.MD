# Create a simple project using Thymeleaf

## Create project with VS Code

```
Ctr + Shift + P
Create Java Project 
# Create a simple project using Thymeleaf
Spring Boot --> Mave Project --> Versoin 3.5.0 --> Java -> Group ID `com.example` --> Artifact Id `PROJECT_NAME` --> Jar --> Version Java (17) -> Dependencies (Spring webMVC+ JPA ....)

```

## Run project

```bash
mvn spring-boot:run
```
